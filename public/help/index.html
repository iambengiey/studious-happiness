FAQs

Where do I update database details?

Set environment variables on your host/server: DATABASE_URL and optional DATABASE_SSL=true. Do not place DB credentials in UI forms or committed files.

How do I confirm DB config is loaded?

Start the server and call authenticated /health. The response includes databaseConfigured.

Can platform admins access all institutions?

Yes, platform_admin is global. Other roles are isolated to a single institution_id.

How are country compliance rules applied?

A country compliance profile is loaded at institution onboarding and auto-applied to message disclaimers, retention, and templates.
